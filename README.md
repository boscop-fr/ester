Ester, le reset CSS
===================

## Installation

`npm install --save @vtech/ester`

## compilons SASS

### Watch

Surveiller les fichiers `.scss` du dossier `scss/` et les recompiler Ã  chaque changement :
`npm run start`

### Build

Regarder les fichiers `.scss` du dossier `scss/` et les compiler une seule fois :
`npm run build`
